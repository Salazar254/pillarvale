# PayFlow MVP

## Setup

1. **Dashboard**: Open `mvp/dashboard/index.html` in your browser.
2. **Widget Test**: Open `mvp/test/merchant-site.html` (requires local server for API calls, or just test UI).
3. **API**:
   - `cd mvp/api`
   - `npm install`
   - `node server.js`

## Structure
- `/dashboard`: Merchant interface
- `/widget`: Embeddable payment widget
- `/api`: Backend for IntaSend integration
